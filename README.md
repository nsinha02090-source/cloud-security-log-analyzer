# Cloud Security Log Analyzer

This project is a command-line tool that analyzes login logs to detect suspicious activities.

## Features
- Detects brute force attacks
- Identifies unknown IP addresses
- Classifies alerts (HIGH, MEDIUM)
- Generates summary report
- Saves report to file

## Technologies Used
- Python
- Colorama

## How to Run
pip install colorama
python analyzer.py logs.txt# cloud-security-log-analyzer
